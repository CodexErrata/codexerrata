# Design Tokens

## Typography

- Headlines: Georgia/Cambria fallback serif now. Replace with licensed GT Sectra, Tiempos or equivalent if available.
- Body: Inter/system sans.
- Letter spacing is only used for small uppercase metadata.

## Color

- `ink`: `#000000`
- `paper`: `#f2e7d8`
- `bone`: `#fffaf2`
- `line`: `#d7bea0`
- `moss`: `#5b5f43`
- `rust`: `#7f3528`
- `slate`: `#3a302a`

The palette avoids gradients and neon. Text remains black or white only; muted color is reserved for surfaces, rules and restrained accents.

## Layout

- Max content width: `1180px`
- Page rhythm: large editorial sections with visible rules
- Cards are used only for repeated archive/contact/KPI items
- Metadata remains visible instead of hidden behind interactions

## Motion

No motion is required for the initial site. Any future motion should be subtle and non-essential.
